# Design

## Instructions
The set of possible operations in ARMv7 is:
- Move
- Arithmetic
  - Add
  - Subtract
  - Multiply
  - Divide
- Count leading '0'
- Compare
- Logical
  - AND
  - OR
  - XOR
  - NOT
  - etc.
- Branch
  - Branch
  - Branch and Link
  - Branch and Exchange
- Stack Operations
  - Push
  - Pop
- Memory Operations
  - Load
  - Store

... and many variations and multiple other less common operations.
